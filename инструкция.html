Инструкции по использованию бота

1. Первоначальная настройка

При первом запуске необходимо ввести номер телефона, код подтверждения и пароль. Это требуется только для первой настройки и не будет запрашиваться снова.

2. Добавление бота в чат

Чтобы активировать команды, добавьте аккаунт бота в свой чат, созданный для настройки. Команды должны отправляться вами напрямую с аккаунта бота в этом чате.

3. Список команд

/prompt

Устанавливает промпт для конкретного пользователя или чата. Вы можете указать имя пользователя для личных сообщений или ID чата, в котором будет использоваться промпт. Также можно использовать флаг g для установки глобального промпта, который применяется, если для чата или пользователя не задан специфический промпт.

Примеры:

/prompt UserName <текст_промпта>

/prompt 123456789 <текст_промпта>

/prompt g <текст_промпта>

/id

Получает идентификатор чата. Команда должна быть отправлена в чат, ID которого вам нужен. Для сохранения приватности сообщение сразу удаляется, делая действие незаметным. ID отправляется в "Избранное" бота для удобного доступа.

Пример:

/id

В "Избранном" вы можете нажать на полученный ID, чтобы скопировать его в буфер обмена. Затем используйте команду /prompt для установки промпта для этого чата.

/deep

Устанавливает глубину контекста сообщений, которые отправляются через API. Принимает два числа в качестве аргументов. По умолчанию глубина истории чата составляет 15, а глубина истории диалога с пользователем – 20.

Пример:

/deep <a> <b>

Где a — количество сообщений общей истории чата, а b — количество пар сообщений в диалоге с пользователем (в формате {сообщение_пользователя : ответ_бота}).
